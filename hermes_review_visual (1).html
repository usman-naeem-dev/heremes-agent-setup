01 — Live Setup Walkthrough

Full Install, Step by Step

1

Get a VPS or use local machine

Any VPS works. Or your own Mac Mini, Linux box, whatever. Hermes runs the same either way.

VPS or Local · Same process
2

Run the one-liner in your terminal

Copy paste. Hit enter. It auto-installs Python, Node, Git — whatever it needs. You don't touch any of that.

curl -fsSL https://get.hermes-ai.com | bash
Under 60 seconds to install
3

OpenClaw detected? Create a NEW bot

Don't say yes to "import from OpenClaw" unless you want your existing bot to break. Learned this the hard way.

⚠️ Using the same bot token as OpenClaw will break everything. Create a fresh Telegram bot via BotFather. Name it something new.
4

Select your LLM provider

Anthropic OAuth doesn't work with Hermes — real downside. Use OpenRouter instead and paste your API key. Pick Kimi K2.

⚠️ Anthropic direct auth is broken in Hermes. Route through OpenRouter if you want Claude. Adds friction + extra cost.
✓ OpenRouter works great. Set default model to Kimi K2 or any model you prefer.
5

Configure settings & Telegram

Working dir → sudo → agent settings → session reset → Telegram token → user ID. If you get lost, ask Claude Code — it explains any prompt instantly.

Pro tip: ask Claude Code to explain any confusing prompt
6

Run hermes — you're in

Type "hermes" in terminal. Send "hi" on Telegram. If it replies — you're live.

✓ "Hi, I'm Hermes, an AI assistant..." — You are alive, friend.
